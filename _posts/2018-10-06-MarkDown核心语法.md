Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。  
Markdown具有一系列衍生版本，用于扩展Markdown的功能（如表格、脚注、内嵌HTML等等），这些功能原初的Markdown尚不具备，它们能让Markdown转换成更多的格式，例如LaTeX，Docbook。  
Markdown增强版中比较有名的有Markdown Extra、MultiMarkdown、 Maruku等。这些衍生版本要么基于工具，如Pandoc；要么基于网站，如GitHub和Wikipedia，在语法上基本兼容，但在一些语法和渲染效果上有改动。  
现在就对核心语法进行演示，我挺烦那么多的文字说明的，当年就是因为废话多一度放弃了HTML，现在开始！
## 标题 ##
```
	# 一级标题 #
	## 二级标题 ##
	### 三级标题 ###
	#### 四级标题 ####
	##### 五级标题 #####
	###### 六级标题 ######
```
演示效果：
> # 一级标题 #
> ## 二级标题 ##
> ### 三级标题 ###
> #### 四级标题 ####
> ##### 五级标题 #####
> ###### 六级标题 ######

## 引用 ##
```
	> 这里是引用内容
```
演示效果：
> 这里是引用内容

## 段落 ##
强制换行直接回车不行，需要打两个空格。

## 分隔符 ##
```
	---
```
演示效果：
> ---
> 这就是分隔符效果，直接打三个小"---"即可。

## 列表 ##
无序列表用"+、-、*……"符号直接写就OK  
有序列表用"1.、2.、3.……"序号直接写就OK  
演示效果：
- 无序1
- 无序2
1. 有序1
1. 有序2

## 超链接 ##
```bash
	文字超链接：[百度](www.baidu.com)
	图片超链接：![图片名称](图片地址)
```
演示效果：
[百度](www.baidu.com)

## 代码段 ##
用键盘ESC下面的在英文输入法下的···即为代码块标识。  
在···后面加上"bash"、"Python"等即为代码类型，影响高亮。  
演示效果：
```bash
	cd D:/alading/adlasd
```
```bash
	printf("这是代码演示\n");
```

## 字体 ##
```bash
	*倾斜*
	**加粗**
	***倾斜并加粗***
	~~文字删除线~~
```
演示效果：
*倾斜*
**加粗**
***倾斜并加粗***
~~文字删除线~~

## 表格 ##
```bash
	- | #用来进行分隔操作
	:   #用来设置对齐方式
```
```bash
	表头1 | 表头2 | 表头3
	:-----|:----:|-----:
	左对齐|居中对齐|右对齐
```
演示效果：
表头1 | 表头2 | 表头3
:-----|:----:|-----:
左对齐|居中对齐|右对齐
